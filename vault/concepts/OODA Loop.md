# OODA Loop

Observe, Orient, Decide, Act. John Boyd's framework for decision-making under competitive pressure.

Referenced in the context of [[Information Sovereignty]] and [[Sovereignty]] more broadly. The key insight: in a competitive environment, getting inside the opponent's OODA loop — cycling through observation, orientation, decision, and action faster than they can — creates decisive advantage.

## Relevance to Statecraft Systems

In the current environment where the meta favors information manipulation, the OODA loop applies to information processing. The capacity to:
1. **Observe** — gather relevant information from the environment
2. **Orient** — interpret it correctly through frameworks like [[Natural Law]] and [[Decidability]]
3. **Decide** — choose action based on [[Structural Tension]] (desired outcome vs. current reality)
4. **Act** — execute with the tools available ([[AI-Native vs AI-Naive]])

...faster and more accurately than competitors or adversaries, is a core survival capability.

AI tools dramatically accelerate the Observe and Orient phases. [[Structural Dynamics]] provides the framework for the Decide phase. Technical capability enables the Act phase.
